# ניווט במערכת חילן פרו

כדי לאפשר למשתמש לנווט את דרכו בין השירותים המגוונים שמציעה מערכת חילן פרו, עומדים לרשותו כלי ניווט אחדים.

---

## 1. תפריטים נגללים

- ממוקמים בראש כל מסך
- מאפשרים מעבר לכל אחד משירותי המערכת
- לפי התפריט שאליו הוא משויך

---

## 2. צלמיות (Icons)

- ממוקמות מתחת לשורת התפריטים הנגללים
- מאפשרות מעבר מהיר לשירותים נפוצים
- הצלמיות המופיעות בכל מסך הן בהתאם לאופיו של מסך זה

---

## 3. ניתוב ישיר

לניווט מהיר באמצעות המקלדת.

### מיקום
- שדות בפינה הימנית התחתונה של המסך
- מוצג: מספר השירות, תת-השירות, ותת-שירות נוסף

### דוגמאות

| ערך | משמעות |
|-----|---------|
| `0001 440 12` | שירות 12, נגררת 440, סמל 1 |
| `0005 ___ 12` | שירות 12, תהליך מונחה 05 |
| `____ 003 30` | שירות 30, תת-שירות 3 |

### צורות שינוי

#### א. שינוי מספר שירות
- מעבר לשירות אחר
- בשירותים עם תת-שירות: ניתן לבקש גם את תת-השירות (למשל "04 10")
- זהה למעבר דרך חלון F2
- נשמרת אפשרות חזרה ב-F3
- **קובץ מדגמי**: ניתן להחליף רק דרך חלון F2

#### ב. שינוי מספר תת-שירות/תת-שירות נוסף
- מעבר למסך אחר במסגרת השירות הנוכחי
- **שונה** משימוש בחלון F2
- אין חזרה לנקודת המוצא

### הערה
ניצול הניתוב הישיר מצריך היכרות עם השירות. למי שאינו מכיר - ניתן להשתמש במעבר רגיל דרך מסכי התפריט.
