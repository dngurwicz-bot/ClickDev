# עיצוב המסכים במערכת חילן פרו

## סוגי מסכים

בכל אחת מהשאילתות שבמערכת חילן פרו מופיעים ארבעה סוגים של מסכים:

### 1. מסך התפריט
- המשתמש מציין את הנתונים שברצונו לראות או את הפעילות שברצונו לבצע
- מגוון האפשרויות שהשאילתא מעמידה לרשותו
- המאפיינים הנקבעים כוללים: זיהוי העובד, תקופת התוקף ועוד
- זיהוי העובד נשמר משאילתא לשאילתא
- ליד מספר העובד מוצג גם שמו

> **הערה**: במסך התפריט לא ניתן להשתמש בשדות הניתוב הישיר למעבר אל מסכי נתונים בתוך אותו שירות - לשם כך יש להשתמש בשדות המיועדים לכך. שדות הניתוב הישיר יכולים לשמש רק למעבר לשירות אחר.

### 2. מסך הנתונים
- המשתמש מקבל את הנתונים שאותם ביקש
- **כותרת אחידה** בכל השאילתות המכילה:
  - שדות הניתוב הישיר
  - שם השירות ושם תת-השירות המוצג
  - מספר המפעל
  - אם נמצאים בקובץ "מדגמי", מוצג מספר הקובץ ליד מספר המפעל
- **שורה מזהה** בכל השאילתות המציגות נתוני עובד:
  - מספר עובד
  - שם פרטי
  - שם משפחה

#### מעברים ממסך נתונים
- מעבר ממסך נתונים אחד למשנהו באמצעות מסך התפריט
- אפשרות לשנות מאפייני מסך הנתונים (החלפת מספר עובד או שמו)
- מעבר מתת-שירות אחד למשנהו באמצעות שדות הניתוב הישיר

> **עיקרון מנחה**: לאפשר ביצוע של מקסימום המעברים בצורה ישירה ממסך הנתונים, ללא צורך בחזרה למסך התפריט.

### 3. חלון תיעוד
- המשתמש מקבל הסבר על האפשרויות העומדות לפניו
- מסך התיעוד המוצג הוא **רגיש הקשר** - מוצג התיעוד הרלוונטי למסך המתאים

### 4. חלון לבחירת נתונים מטבלה
- מקשים אחדים מאפשרים לקבל חלונות לבחירת נתונים מטבלה
- כולל:
  - בחירת תהליך מונחה
  - בחירת-סוג נתון
  - ועוד
