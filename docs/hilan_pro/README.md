# תיעוד מערכת חילן פרו

תיקייה זו מכילה את התיעוד המלא של מערכת "חילן פרו" - מערכת לניהול משאבי אנוש, שכר ונוכחות.

> **הערה חשובה**: כרגע אנחנו בונים רק את מודול **משאבי אנוש**, לא שכר ולא נוכחות.

## שירותים עיקריים

1. **הצגת מידע ארגוני** - טבלאות כלליות ומפעליות
2. **תיק העובד** - צפייה ועדכון נתוני עובד
3. **שאילתות פרט** - עיון בנתוני עובד בודד
4. **שאילתות קיבוציות** - טיפול בקבוצות עובדים (מחולל "אופק")
5. **תוכניות שירות** - תחזוקת המערכת

---

## עקרונות מנחים

- הבאת כל המידע לקצות אצבעות המשתמש
- פשטות תפעול עם סיוע רגיש-הקשר
- צמצום הצורך בדו"חות מודפסים
- המחשב מסביר את החלטותיו
- שיפור היכולת לטיפול מיידי ושלם במשימות
